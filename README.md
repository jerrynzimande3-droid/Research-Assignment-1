This assignment provided an overview of key concepts in data analytics and their application in modern organisations. It explored foundational topics such as databases, data warehouses, and data lakes, highlighting how data is stored, managed, and prepared for analysis. The assignment also examined processes like ETL (Extract, Transform, Load), which are essential for transforming raw data into meaningful insights.
In addition, the assignment covered analytical tools and techniques, including data mining, machine learning, and data modelling, explaining how businesses use these to identify patterns, predict outcomes, and support decision-making. The role of APIs and data formats such as JSON, CSV, Parquet, and Avro was also discussed, demonstrating how data is exchanged and integrated across systems.
Furthermore, the assignment introduced concepts related to artificial intelligence, including deep learning and natural language processing, with practical examples of how these technologies are used in everyday applications and business environments.
